# Install LODA on iOS


## Prerequisits

###  Install the `iSH` app.


### Install GCC

```
PROMPT> apk add build-base
```

### Install OpenSSH

```
PROMPT> apk add openssh
```

### Install GIT

```
PROMPT> apk add git
```

### Install cURL

```
PROMPT> apk add curl
```

### Install rust

You can skip this step. It's only needed for "loda-rust".

```
PROMPT> apk add rust
```

### Install cargo

You can skip this step. It's only needed for "loda-rust".

```
PROMPT> apk add cargo
```
